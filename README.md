# for-mqttc

This is an MQTT client desktop tool that facilitates users in connecting to a broker and subscribing to and publishing topics.It is developed based on Tauri and Tailwind.


Support function:
1. protocal version: v3(v4),v5
2. tls server auth
3. tls client auth
